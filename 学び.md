三項演算子
？　はif
：はelse
として使用できる

例
if (りんごの数 == 0) {
  return 'うれしい！';
} else { return 'ざんねん！'; }

と

return (りんごの数 == 0) ? 'うれしい！' : 'ざんねん！';

は同じ意味を持つ
